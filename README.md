# 使用方法

1. `git git@github.com:DENGWENPENG/cet4.git`
2. `cd cet4 `
3. `virtualenv venv `
4. `cd venv\Scripts\  && activate`
5. `pip install -r requirements.txt `
6. `python app.py `


# 打包
## 进入虚拟环境打包，生成的exe文件较小（重复使用方法的 4）
1. `pyinstaller -F app.py `
